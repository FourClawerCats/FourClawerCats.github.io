3.常量和变量
TZ Lv5
 2016-03-03 21:10:21  
  • java 
 
  • java基础 
 374 Words  1 Mins

常量:在程序运行中,值不会发生改变的量叫做常量
变量:在程序运行中, 值会发生改变的量叫做变量

变量:

  • 先声明 后赋值
  • 声明 + 赋值

注意:

  • 在类内, 方法外定义的变量叫成员变量 有默认值

  • 在方法内, 定义的变量必须进行初始化操作

  • java是一种强类型语言,每个变量都必须声明类型

  • java变量是程序中最基本的存储单元,其要素包括, 变量名, 变量类和作用域

  • 变量在使用前必须对其声明, 只有在变量声明之后, 才能为其分配相应的长度存储单元

  • type varName = value

注意事项

  • 每个变量都有类型, 类型可以是基本类型, 也可以是引用类型
  • 变量名必须是合法的标识符
  • 电脑使用内存来记忆计算时锁使用的数据,人类采用旅馆来才能出外出住宿的人们
  • 内存中的变量好比旅馆的房间, 内存中的常量好比住旅馆房间的人

final 常量

  • 使用 final 修饰的变量, 只能被初始化一次, 成为了常量
  • final常量是有名称的
  • 是不可修改的
