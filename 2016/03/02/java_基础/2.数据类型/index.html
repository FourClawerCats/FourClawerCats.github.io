基本数据类型

java是一种强类型语言

  • 强类型表示, 变量在定义的时候必须显式的声明类型是什么
  • 弱类型表示, 变量会根据值自己去推断, 不需要指定类型是什么
  • 常量是有数据类型的
  • 变量必须声明其数据类型

数据类型

  • 基本数据类型 四类八种
    • 数值型
      • 整数类型(byte, short, int, long)
        • byte: 使用一个字节存储, 因此范围是: -128 到 127
        • short: 使用两个字节存储, 因此范围是 -32768 到 32767
        • int: 使用四个字节存储, 因此范围是 正负21亿
        • long: 使用八个字节存储, 因此范围是…..
      • 浮点类型(float, double)
        • float: 单精度 四个字节
        • double: 双精度 八个字节
    • 字符型(char)
      • 占两个字节的时候,用单引号表示 ‘’
    • 布尔型(boolea): 只有trun || falst
  • 引用数据类型
    • 类 (class)
    • 接口 (interface)
    • 数组

整形常量

  • 10进制整数
  • 8进制整数, 要求以0开头, 如015
  • 16进制数, 要求0X或0X开头, 如0x15

整形变量

  • 整形常量默认为 int型, 声明long型常量可以后面加L
  • long a = 555555555; 不出错, 在Int表示的范围内 21亿
  • long b = 5555555555555555555; //不加L出错, 已经超过int表示的范围了
  • long 类型 使用时候没有超过 21亿 没有超过int长度 则不加L 也不会报错
  • 在使用数值的时候默认都是int类型

浮点类型常量

  • 十进制数形式 1.01 0.33
  • 科学计数法 314E2 314E-2

浮点类型变量

  • Float类型: 单精度类型, 尾数可以精确到七位有效数, 很多情况下,Float剋行的精度很难满足需求
  • double类型: 双精度类型 精度是float的两倍 绝大部分的应用程序都采用duouble 类型

注意

  • 浮点常量more为double, 要变为float, 需在后面增加F 如:3.14F
  • 浮点数存在舍入误差, 很多数字不能精确表示, 如果需要不产生舍入误差的数字计算, 需要使用BigDecimal类

字符数据类型

  • 单引号用来表示字符常量. 例如’A’是一个字符, 他与”A”是不同的. “A”表示含有一个字符的字符串
  • char类型 用来表示在Unicode编码表中的字符,所以在运算时可以被当作一个整形来处理
  • char类型在内存中存储的是该字符的Unicode编码值, 所以char类型可以当作int类型来处理
