标识符Identifier

作用

  • 常量,变量,方法,类和包等起名字 名称

命名规则

  • 必须以字母,下划线,美元符号开头
  • 其他部分可以是字母 下划线, 美观符号 和 数字的任意组合
  • 大小写敏感, 长度无限制
  • 不可以是java的关键字

注意

  • java不采用通用语言使用的ASCII字符集
  • java采用unicode这样的标准国际字符集,因此,这里的字母的含义:英文,汉字,等等,不要用汉字来定义标识符!

标识符命名规范

硬性规定

  • 标识符必须以字母,下划线或美元符号开头
  • 其他部分必须是字母,数字,下划线或美元符号,但不能出现特殊字符
  • 标识符大小写敏感
  • 不能是java的关键字或保留字 (留给系统使用的表示特殊含义的字符串)

常规建议

  • 驼峰标识
    • 类名, 接口名称在命名的时候要首字母大写
    • 方法, 变量命名的时候首字母要小写
    • 多个单词拼接表示一个标识符时候, 每个单词的首字符都大写
  • 见名知意
    • 通过标识符的名称能知道其表达的含义
    • 千万不要写拼音

关键字/保留字

  • java关键字时java语言保留提供内部使用的,如class用于定义类,关键字也可以称之为保留字,他们的意思是一样的
