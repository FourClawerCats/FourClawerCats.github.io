启用

sudo ufw enable
sudo ufw default deny
运行以上两条命令后，开启了防火墙，并在系统启动时自动开启。关闭所有外部对本机的访问，但本机访问外部正常。

关闭

sudo ufw disable

开启/禁用

sudo ufw allow 22/tcp 开启
sudo ufw deny 22/tcp 禁止外部访问
sudo ufw delete allow 22/tcp 删除上面建立的某条规则

查看防火墙状态

sudo ufw status
