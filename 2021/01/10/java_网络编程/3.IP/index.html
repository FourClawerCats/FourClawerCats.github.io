IP地址: InetAddress

  • 唯一定位一台网络上的计算机
  • 127.0.0.1 : localhost

IP地址分类

  • 公网 (互联网)

    • ABCD类地址
      • 0
      • 128
      • 192
      • 255

  • 私网 (局域网)

    • 192.168.0.1

  • IPV4

    • 127.0.0.1 , 四个字节组成, 0-255, 42亿;
    • IPV4 三十亿在北美, 亚洲只有四亿;

  • IPV6

    • fe80::b509:2f5f:1fa3:cd0b%22, 128位 八个无符号整数

    • 2021 : 1ade : 3333 : 0023 : 0000 : 0000 : 1111 : 3333

  • 域名: 解决记忆IP的问题;
