请你谈谈对Volatile的理解

volatile是 java虚拟机提供轻量级的同步机制

  • 保证可见性
  • 不保证原子性
  • 禁止指令重排

什么是JMM?

  • java 内存模型
  • 不存在的东西
  • 是一种概念
  • 是一种约定

关于JMM的一些同步的约定

  • 线程解锁前, 必须把共享变量立刻刷回主存
  • 线程加锁前, 必须读取主存中的最新值到工作内存中
  • 加锁和解锁是同一把所

JMM 八种操作

  • read 读取
  • load 加载
  • Use 使用
  • assign 赋值
  • store 存储
  • write 写入
  • lock 加锁
  • unlock 解锁

不允许规则单独出现.
使用了存储就必须写入, 其他规则也一样, 相互关联.
